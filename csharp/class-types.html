1. Sealed Class
    It cannot be inherited from another class.
  2. Static Class
    A static class in C# is a class that cannot be instantiated. A static class can only contain static data members including static methods, static constructors, and static properties.
    Those are useful and provide an easy way to access its members that does not need to work differently for different objects.
    Eg: Math
  3. Abstract Class
    An Abstract class that provides a common definition to the subclasses.
    This is a type of class whose object is not created.
