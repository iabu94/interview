• These 3 are define object creation process in .NET Core DI when multiple objects of the same type have to be injected
  • In transient, new object instances will be injected in a single request and response. In other words to every controller and every service, a new instance will be provided.
  • In scoped, the same object instance will be injected in a single request and response.
  • In singleton, the same object will be injected across all requests and responses. In this case one global instance of the object will be created.
